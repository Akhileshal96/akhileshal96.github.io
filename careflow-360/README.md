# CareFlow 360

CareFlow 360 is a responsive clinic, hospital, and multi-facility operations MVP with mobile-ready daily, weekly, monthly, and yearly insights.

## Included

- First-time facility mode selection
- Clinic, hospital, and multi-facility dashboards
- Responsive desktop and mobile layouts
- Progressive Web App manifest and offline shell caching
- Patient list and search
- Task, priority, and SLA tracking
- Operational module overview
- Daily, weekly, monthly, and yearly insights
- Smart recommendations
- Downloadable JSON summaries
- Local browser storage for the selected facility mode

## Local run

Use any static web server from this folder, for example:

```bash
python3 -m http.server 8080
```

## Important

This is a front-end MVP with fictional sample data. It is not a production clinical system and must not be used to store real patient-identifiable information. A production version requires secure authentication, backend APIs, encrypted databases, audit logging, role-based access controls, backups, consent workflows, integrations, and healthcare compliance review.
